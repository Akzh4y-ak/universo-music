## Content Optimization Checklist

### On-Page SEO

- [ ] **Title Tags (50-60 chars)**
  - Format: "Keyword | Sub-keyword | Brand"
  - Example: "Free Music Streaming Online No Sign-up | Univerzo"

- [ ] **Meta Descriptions (150-160 chars)**
  - Include primary keyword + benefit + CTA
  - Example: "Listen to millions of songs free with no sign-up. Instant access to trending music and playlists. Stream online now."

- [ ] **H1 Tag (1 per page)**
  - Must include primary keyword
  - Should be different from title tag

- [ ] **H2/H3 Tags**
  - Use variations of primary keyword
  - Create topic clusters

- [ ] **Body Content (1500+ words optimal)**
  - Natural keyword placement (2-3% density)
  - Featured snippet answer (60-100 words)
  - Clear section headings
  - Lists and tables for scannability

- [ ] **Images**
  - Alt text includes keywords
  - Descriptive file names
  - Compressed for speed
  - Thumbnails for all content

- [ ] **Internal Links** (8-12 per page)
  - Relevant anchor text
  - Contextual placement
  - Mix of deep and shallow links

- [ ] **External Links**
  - Link to authority sites (schema.org, Google)
  - Relevant domains only
  - 3-5 per major section

### Technical SEO

- [ ] **Page Speed** (Target: <3 seconds)
  - Lazy load images
  - Minimize CSS/JS
  - Enable GZIP compression

- [ ] **Mobile Responsive**
  - Test on all devices
  - Touch-friendly buttons
  - Readable text sizes

- [ ] **Core Web Vitals**
  - LCP: <2.5s
  - FID: <100ms
  - CLS: <0.1

- [ ] **Structured Data**
  - Schema.org markup (JSON-LD)
  - Rich snippets
  - Breadcrumbs

- [ ] **Sitemap**
  - XML sitemap submitted
  - All pages included
  - Regular updates
